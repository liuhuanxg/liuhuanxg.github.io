    equals()：作用也是判断两个对象是否相等。但它一般有两种使用情况。

    • 情况1：类没有覆盖equals()方法。则通过equals()比较该类的两个对象时，等价于使用”==”比较。
    • 情况2：类覆盖了equals()方法。一般，我们覆盖equals()方法来比较两个对象的内容是否相等；若他们的内容相等，则返回true(即认为两个对象相等)。
